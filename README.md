# 情報工学科の特徴
1. 自分のやりたいことに合わせた**5つのコース**
情報工学科では自分の興味に合わせて、２年次にコース選択を行うことができます。
コース選択では
* システムエンジニアリングコース
* インターネット＆セキュリティコース
* 知能情報コース
さらに、デザイン学科とのコラボレーションコースでは
* ビジュアルコンピューティングコース
* Webデザインコース
から選択が可能です。
2. 様々なジャンルの**研究室**
情報工学科では多くの研究室が存在し、そこで様々な研究を行うことができます。
[研究室](https://feng.takushoku-u.ac.jp/course/cs/lab.html)
